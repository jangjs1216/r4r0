# AuthViewService

- **책임**: 세션 상태(인증/잠금), MFA 여부, API 키 목록, 사용자 정보를 주어진 컨트롤과 함께 표시.
- **계약(입력 DTO)**: `contracts/frontend/auth.schema.json` (session, user, controls[]).
- **경계**: 인증/키 회전을 직접 수행하지 않으며 DTO만 렌더링; 명령은 오케스트레이터/백엔드 API를 통해야 함.
- **상호작용**: 컨트롤 클릭 시 로컬 처리 대신 오케스트레이터로 의도(예: 키 회전, 트레이딩 잠금)를 전달.
- **테스트**: 계약 검증, 배지/키 렌더 스냅샷, 컨트롤 의도 전파 확인.
